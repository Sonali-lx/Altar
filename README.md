# Project Altar ✝️  
**Proverbs 16:3 – Commit your works to the Lord, and your thoughts will be established**

---

## 📖 Vision
Project Altar is a spiritual growth platform designed to help believers deepen their personal relationship with God, strengthen fellowship with others, and support committee work. It is inspired by the idea of building a digital altar — a place for devotion, reflection, and service.

---

## ✨ Features (Planned)
- **Bible Meditation**
  - Quiet Time (ASPECT method: About God, Sins to Avoid, Promises to Claim, Examples to Follow, Commands to Obey, Theme of Passage)
  - Freeform entry (“What did God speak to me today?”)
  - Personal Bible Study (PBS: Observation, Interpretation, Application)
  - Group Bible Study (GBS: 6-step method)
- **Quiet Time Streak Tracker**
- **Bible + Journal side by side**
- **Verse of the Day**
- **Prayer Journal**
  - With format (Adoration, Confession, Thanksgiving, Supplication)
  - Without format (freeform prayers)
- **General Journal Page** (thoughts, findings, reflections)
- **Community**
  - Quiet Time Sharing Forum
  - Bible Discussion Forum
- **Committee Tools**
  - Minutes of meetings
  - Treasurer wallet
  - Upcoming meetings
  - Program planning + follow-ups
- **Reminders**
  - Quiet time reminders
  - Event reminders
- **Prayercells**
  - Members, Leaders, Parents
- **Offering Page**
- **Future Enhancements**
  - ML model for personalization
  - AI chatbot trained on Scripture
  - Deployment on AWS

---

## 🛠 Tech Stack
- **Frontend**: React (SPA with React Router)  
- **Backend**: Node.js + Express  
- **Database**: MongoDB (Mongoose ODM)  
- **Deployment**: AWS (planned)  

---

## 🚀 MVP (Phase 1)
- Bible Meditation (Quiet Time + Freeform)  
- User model (basic roles: student, committee member, prayercell leader/parent)  
- Routes + Controllers for meditation entries
